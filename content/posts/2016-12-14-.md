&#8220;Can you do it?&#8221; &#8211; and here I was, put in a position that many of us fear but want to almost immediately answer with &#8220;Sure.&#8221;, and so I did. &#8220;Sure.&#8221; I said, knowing that I do not yet possess the knowledge needed to create the application the client wanted me to create. I&#8217;d recently read about Electron and how it&#8217;s all Javascript and NodeJS and all the unicorns in the world are riding it right now and whatnot.

Let&#8217;s start with anything except the beginning &#8211; this is how I learn anything because I usually can&#8217;t stand to read the &#8220;so, you write _echo &#8220;Text here&#8221;_ to show text&#8221; part of any programming language. I always felt that I learn new languages easier if I start a project and learn things as I go and as I need them.
